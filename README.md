# How to Run — Dell Inventory Management System

## What You Need

Just a web browser (Chrome, Edge, Firefox, etc.). No installation, no Node.js, no server needed for this part.

## Files Required

Make sure these 3 files are together in the **same folder**:
- `index.html`
- `style.css`
- `script.js`

If they're not in the same folder, the page will open but it will look unstyled and the buttons won't work.

## Steps to Run

1. Download/save all 3 files into one folder (e.g. a folder called `inventory-app`).
2. Open that folder.
3. Double-click `index.html`.
4. It will open automatically in your default web browser.

That's it — the app is now running.

## How to Use It Once It's Open

- **See current stock:** Scroll to "Current Inventory" — it lists all products already loaded.
- **Add a product:** Type a product name in the "Add Product" box, then click the **Add Product** button.
- **Check stock:** Type a product name in the "Check Availability" box, then click **Check Stock**. It will tell you if that product is in stock or out of stock.
- **Remove a product:** Click the **Remove** button next to any product in the list.

## If It Doesn't Open Properly

- Make sure the file is named exactly `index.html` and you're double-clicking that one (not `style.css` or `script.js`).
- Make sure all 3 files are in the same folder — don't move just one of them elsewhere.
- If it opens but looks plain/unstyled, the browser likely can't find `style.css` — double check it's in the same folder as `index.html`.
- If buttons don't respond, right-click the page → Inspect → Console tab, and check for a red error message (this usually means `script.js` isn't in the same folder).
